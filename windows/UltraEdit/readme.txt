UltraEdit32 - v11.10a 简体中文版

能够满足你一切编辑需要的编辑器。

UltraEdit 是一套功能强大的文本编辑器，可以编辑文本、十六进制、
ASCII 码，可以取代记事本，内建英文单字检查、C++ 及 VB 指令突显，
可同时编辑多个文件，而且即使开启很大的文件速度也不会慢。软件附
有 HTML 标签颜色显示、搜寻替换以及无限制的还原功能，一般大家喜
欢用其来修改EXE 或 DLL 文件。

运行环境:
Windows 98/Me 和 Windows NT/2000/XP/2003。

标准功能:
 - 基于磁盘的文本编辑器
 - 文件大小不限, 同时打开多个数 MB 的文件，RAM 的占用率也很少
 - 同时打开并显示多个文件
 - 列块模式编辑! 以列块方式插入、删除、剪切一或数个连续的列
 - 可拖拽编辑
 - 文件排序 (删除重复, 忽略大小写, 递增, 递减)
 - 100,000 字拼写检查
 - 语法加亮 - 对于 C/C++, VB, HTML 和 Java 可配置, 预配置
 - 文字在以硬回车为标记的指定列自动换行
 - 在文档的光标位置插入一个文件
 - 支持从文件管理器拖拽
 - 工具栏可配置
 - 窗口可拆分
 - 插入和覆盖编辑方式
 - 多级撤销和重做
 - UltraEdit 是适用于 Windows 3.x CUA 的程序
 - 查找和替换 - 也允许按住 Shift 键，在脱字符和查找目标之间选
   择文本, 替换所有选区中的内容
 - 在文件中查找, 在文件中替换
 - 转到某行或某页
 - 显示和打印机的字体选择。(支持包含 TRUE-TYPE 字体的所有已安
   装字体)
 - 打印支持页眉, 页脚, 页边距和页号
 - 行自动缩进
 - 制表站设置
 - 支持文字重排
 - 十六进制编辑器 - 允许编辑任何二进制文件 - 支持 Hex 剪切, 复
   制和粘贴
 - HEX 字符的插入和删除
 - HEX 查找, 替换和全部替换
 - 书签
 - 多个窗口打开同一个文件
 - 全面支持宏命令, 包括保存和加载宏
 - 帮助上下文敏感
 - 在原始文件目录下自动生成扩展名为 (.BAK) 的备份文件
 - 每次使用 UltraEdit 时，它将保持它原来的屏幕位置
 - 行号/列号显示 (行号显示可以禁用)
 - 鼠标右键弹出菜单
 - 文本转换为小写或大写和大写
 - Unix/Mac 转 DOS
 - DOS 转 Unix
 - 自动检测 UNIX/Mac 文件
 - 转换文字换行为 CR/LF，允许用硬回车文字换行的方式写入到文件
 - 转换 CR/LF 为文字换行 (从文件中删除 CR/LF)
 - 支持模板
 - 更多 ...
 并且 UltraEdit 还接受命令行方式输入，因此它能够替代笔记本或其
 它编辑器。
